# chart

Real-time charts for Clojure

## Usage

e.g.
Define some functions that take no args:

    (defn rand1 [] (rand 1))

(maybe your functions will deref an atom or similar)

then:
    (show (time-chart [rand1 rand1]) :title "test random funcs")
